Question 3: Rules

Write rules to describe the following relations:

  • parent(Parent,Child)
  • son(Son,Parent)
  • daughter(Daughter,Parent)
  • grandfather(Grandfather,Grandchild)
  • grandparent(Grandparent,Grandchild)
  • brother(Brother,Sibling)
  • sibling(Sibling1,Sibling2)
  • have_children_together(Person1,Person2)
  • uncle(Uncle,Person)

Check if half-brothers are considered brothers. If so, change the predicate in such a way that this is not the case anymore. Can a person be his own brother? Yes, fix the problem.

Do the same for the sibling relation. Are halfbrothers/halfsisters siblings of each other? Yes, fix the problem. Can somebody be a sibling of hisself? Yes, fix the problem.

Question 5: Anonymous Variables

You can use “_” (underscore) as a name for a anonymous variable. Every occurence of “_” is considered as a new variable, so you cannot use “_” to write the grandfather relation as: 

grandparent(X,Z):-parent(X,_), parent(_,Z).

Use “_” to solve the following problems:

  • write a query to check if saar has a child
  • write a universal quantified predicate human
  • define a predicate isparent(Person) that check if Person has at least one child

Question 6: Second order predicate

findall(Var,Goal,Bag) creates a list of the instantiations Var gets successively on backtracking over Goal and unifies the result with Bag. It succeeds with an empty list if Goal has no solutions. findall/3 is equivalent to bagof/3 with all free variables bound with the existential operator (^), except that bagof/3 fails when goal has no solutions.

bagof(Var,Goal,Bag) unifies Bag with the alternatives of Var. If Goal has free variables besides the one sharing with Var, bagof will backtrack over the alternatives of these free variables, unifying Bag with the corresponding alternatives of Var. The construct +Var^Goal tells bagof not to bind Var in Goal. bagof/3 fails if Goal has no solutions.

setof(Var,Goal,Bag) is equivalent to bagof/3 but it sorts the results using sort/2 to get a sorted list of alternatives without duplicates.

Write queries to:

  • find all children of bernard?
  • find all sons of jean?
  • check if albert and simonne have only children together?
  • check if all your data is consistent, i.e. fathers are male, mothers are female.

Question 9: Monkey/Banana riddle

Solve the following problem by writing a prolog program.

“A monkey is sitting by the door of the room. In the middle of the room there is a banana hanging on the ceiling, but the hungry monkey cannot grab it because it is too high. But, there is a box standing at the window that the monkey can use to get the banana. The monkey can push the box when he is near to it and he can climb on it when he is near to it. When the box is under the banana and the monkey is standing near the box, he can climb on the box and grab the banana.”
